# Simple Number Counter

A small JavaScript project I made while learning JavaScript that lets you increase or decrease a number with buttons.

## How to Use

1. Open `index.html` in your browser
2. Click the **Increase** button to increase the number  
3. Click the **Decrease** button to decrease the number
4. Click the **Reset** button to reset the number

## Features

- Increment and decrement numbers
- Simple UI
